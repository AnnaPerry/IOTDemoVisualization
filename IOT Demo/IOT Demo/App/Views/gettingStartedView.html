This app displays live sensor data from a PI System that is installed on an IoT gateway. The app collects that data from sensors on your phone and writes that data to the gateway PI System.
A pop-up will instruct you to move your device to create interesting motion data. Click “Close” to exit the pop-up begin using the app.
The schematic shows sensors on your device read by this app. The chart shows live values from your device that were written to the gateway.
Click the “Table” button on the navbar to see a table list of raw sensor values and calculations from the gateway PI System.
Click the “Chart” navbar button to see how sensor values change over time. Rotate and move your device to generate new values for the chart.
Click “Asset Dashboard” to see top-level KPIs, or click “Assets” to see a list of data sources connected to the gateway.
Click “Getting Started” to re-open this tutorial, see support information, and more. Click “Asset 1” button to return to Asset 1, or click another asset.
Some gateways receive data from WISE 4012 wireless IoT sensor modules. In the Assets View, click “WISE-4012E 1” to see live data from this module.
Rotate a WISE 4012 input board potentiometers to raise or lower the Analog Input voltage values, or toggle its switches to change the Digital Input values from low to high.
Click to get started using this web app! You’ll next see a live visualization of sensor data from your phone.
